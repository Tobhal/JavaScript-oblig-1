What is the modulo function used for in programming (task 5)

The modulo function is usefull when:

  • - You need to figure out if a number is odd or even.
    • You take your number modulo 2
    • This will give you 1 if the number is odd
    • 0 if the number is even.
  • - You want syckle between a range of numbers with a single rising number.
    • Eksample for this is a clock. When the system clock number goes up you want to just show the seconds (0 - 59)
    • For this will work you will take the system time (t) and modulo 60. this will return a number from 0 - 59.

    • Can also do this with arrays (2D). For going thru "line by line".
    • You take a number modulo the lenght, and a number for knowing when you hit 0 again.
    • This will then loop thru 0->array lenght and when it hits 0 aganin you change the line.
